# Satellite Datasets for Hydro-Alert

## Precipitation
- **GPM IMERG:** NASA GES DISC ([access](https://disc.gsfc.nasa.gov/datasets/GPM_3IMERGDF_06/summary))
    - Format: NetCDF4
    - Variable: precipitationCal

## Digital Elevation Model (DEM)
- **SRTM DEM:** AWS Open Data ([access](https://registry.opendata.aws/srtm/))
    - Format: GeoTIFF or HGT

## Flood Extent Products
- **MODIS:** NASA FIRMS ([access](https://firms.modaps.eosdis.nasa.gov/))
- **Sentinel-1 SAR:** Copernicus Open Access ([access](https://scihub.copernicus.eu/))
